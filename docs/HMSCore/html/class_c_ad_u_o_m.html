Mô tả chi tiết

View quản lý đơn vị tính (Unit of Measure - UOM).

Class này kế thừa từ CGuiView, cung cấp giao diện để quản lý danh sách đơn vị tính:

  • Hiển thị danh sách đơn vị tính
  • Tìm kiếm và lọc đơn vị tính
  • Thêm, sửa, xóa đơn vị tính
  • Quản lý các bộ lọc tìm kiếm
