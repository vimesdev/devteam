Mô tả chi tiết

Popup dialog nhập đơn thuốc ung thư (phiên bản mở rộng).

Class này kế thừa từ CGuiPopup, cung cấp các chức năng:

  • Nhập đơn thuốc ung thư trong popup window
  • Chọn thuốc, đơn vị, số lượng, chỉ định
  • Quản lý tốc độ truyền và cách dùng
  • Phiên bản mở rộng với các tính năng bổ sung cho điều trị ung thư
