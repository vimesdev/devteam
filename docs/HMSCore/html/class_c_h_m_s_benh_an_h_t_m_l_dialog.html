Mô tả chi tiết

Dialog quản lý bệnh án điện tử dạng HTML với nhiều tab.

Class này kế thừa từ CGuiDialog, cung cấp các chức năng:

  • Quản lý bệnh án điện tử dạng HTML với nhiều tab
  • Tích hợp CHMSBenhAnMat1 và CPhanKhamChuyenKhoaHTML
  • Thiết lập và cập nhật bệnh án
  • Lưu và in bệnh án
  • Quản lý thời gian điều trị
  • Nhập form và in form con
  • Quản lý nhiều form động
