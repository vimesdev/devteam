Mô tả chi tiết

Lớp view tích hợp Microsoft WebView2 để hiển thị web content trong ứng dụng.

Class này kế thừa từ CView, cung cấp WebView2 integration:

  • Hiển thị web content sử dụng Microsoft Edge WebView2
  • Navigate đến URL hoặc HTML content
  • Execute JavaScript và nhận kết quả
  • Gửi/nhận messages giữa ứng dụng và web content
  • Hỗ trợ navigation events và process failure handling
  • Inject script vào web page
  • Login và authentication
  • Clear cache và quản lý cookies
  • High DPI awareness
  • Static methods để khởi tạo environment và utility functions
