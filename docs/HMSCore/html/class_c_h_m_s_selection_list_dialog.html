Mô tả chi tiết

Dialog chọn danh sách từ một danh sách có sẵn.

Class này kế thừa từ CGuiDialog, cung cấp dialog lựa chọn:

  • Hiển thị danh sách các item để chọn
  • Chọn nhiều item từ danh sách
  • Quản lý danh sách đã chọn riêng biệt
  • Chọn tất cả hoặc bỏ chọn tất cả
  • Lấy thông tin item đã chọn theo index
  • Hỗ trợ checkbox để enable/disable chức năng chọn
  • Load dữ liệu từ SQL query hoặc JSON
