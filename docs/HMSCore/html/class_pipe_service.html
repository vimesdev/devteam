Mô tả chi tiết

Lớp dịch vụ quản lý named pipe để giao tiếp với các ứng dụng khác.

Class này cung cấp dịch vụ named pipe:

  • Tạo và quản lý named pipe server
  • Xử lý các kết nối từ client
  • Gửi/nhận messages qua pipe
  • Quản lý thread để xử lý các kết nối đồng thời
  • Hỗ trợ giao tiếp với các thiết bị ngoại vi như card reader, fingerprint scanner
