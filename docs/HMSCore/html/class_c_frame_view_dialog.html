Mô tả chi tiết

Lớp mini frame dialog để hiển thị CGuiView trong mini frame window.

Class này kế thừa từ CGuiMiniFrame, cung cấp mini frame:

  • Tạo và hiển thị mini frame với CGuiView
  • Tự động resize view khi frame thay đổi kích thước
  • Quản lý lifecycle của view
