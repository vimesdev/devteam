Mô tả chi tiết

Dialog thiết lập mức độ stent.

Class này kế thừa từ CGuiDialog, cung cấp các chức năng:

  • Chọn mức độ stent từ 1 đến 5
  • Áp dụng mức độ stent cho order và product
  • Quản lý class của stent
