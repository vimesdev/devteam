Mô tả chi tiết

Dialog thiết lập bệnh án điện tử

Class này kế thừa từ CGuiDialog, cung cấp các chức năng:

  • Quản lý các loại bệnh án điện tử
  • Quản lý các bản ghi bệnh án theo loại
  • Quản lý phân quyền theo khoa
  • Thêm, sửa, xóa loại và bản ghi
  • Thiết lập chi tiết cho từng bản ghi
  • Cấp quyền cho khoa
