Mô tả chi tiết

View hiển thị danh sách đơn hàng vật tư/thuốc của khoa.

Class này kế thừa từ CGuiView, cung cấp các chức năng:

  • Hiển thị danh sách các đơn hàng vật tư/thuốc
  • Tìm kiếm và lọc theo khoảng thời gian, kho
  • Quản lý các trạng thái: tất cả, mở, đã gửi, đã duyệt
  • Xem chi tiết đơn hàng
