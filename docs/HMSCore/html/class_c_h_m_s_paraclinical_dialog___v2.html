Mô tả chi tiết

Dialog quản lý đơn cận lâm sàng (phiên bản 2).

Class này kế thừa từ CGuiDialog, cung cấp các chức năng:

  • Tạo và quản lý đơn cận lâm sàng
  • Chọn khoa thực hiện
  • Thêm, xóa các item cận lâm sàng
  • Tự động in đơn
  • Quản lý ngày đặt và bác sĩ
  • Phiên bản 2 với các tính năng cải tiến
