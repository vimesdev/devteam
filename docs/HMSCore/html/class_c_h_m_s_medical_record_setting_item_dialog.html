Mô tả chi tiết

Dialog thiết lập các item trong bệnh án điện tử

Class này kế thừa từ CGuiDialog, cung cấp các chức năng:

  • Quản lý các item trong bệnh án điện tử
  • Load và hiển thị các form HTML
  • Copy item từ form khác
  • Thêm, sửa, xóa các item
  • Lưu và cập nhật JSON data
  • Quản lý list các item với HTML layout control
