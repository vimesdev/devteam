Mô tả chi tiết

Lớp dialog wrapper để hiển thị CGuiView trong dialog.

Class này kế thừa từ CGuiDialog, cung cấp wrapper:

  • Chứa một CGuiView bên trong dialog
  • Tự động resize view khi dialog thay đổi kích thước
  • Quản lý lifecycle của view
