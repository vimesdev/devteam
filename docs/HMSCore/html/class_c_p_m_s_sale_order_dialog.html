Mô tả chi tiết

Dialog quản lý đơn bán hàng (sale order).

Class này kế thừa từ CGuiDialog, cung cấp các chức năng:

  • Tạo và quản lý đơn bán hàng
  • Quản lý thông tin khách hàng và đơn hàng
  • Thêm, sửa, xóa các dòng trong đơn hàng
  • Xác nhận và duyệt đơn hàng
  • In phiếu giao hàng và phiếu GLS
  • Quản lý thanh toán và trạng thái đơn hàng
  • Hỗ trợ thực phẩm chức năng
  • Liên kết với hệ thống Dược Quốc Gia
