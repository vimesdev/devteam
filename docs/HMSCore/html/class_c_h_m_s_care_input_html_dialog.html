Mô tả chi tiết

Dialog nhập thông tin chăm sóc dạng HTML.

Class này kế thừa từ CGuiDialog, cung cấp các chức năng:

  • Nhập thông tin chăm sóc bệnh nhân dạng HTML form
  • Quản lý nhiều loại và sub-type form chăm sóc
  • Load thông tin cơ bản, chẩn đoán, khoa phòng giường
  • Load thông tin phẫu thuật và thông tin mở rộng
  • Copy từ lần điều trị trước
  • In form chăm sóc
  • Quản lý y lệnh bác sĩ và lịch sử
