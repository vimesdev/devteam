Mô tả chi tiết

Lớp mini frame chứa WebView2 để hiển thị web content trong cửa sổ riêng.

Class này kế thừa từ CGuiMiniFrame, cung cấp cửa sổ mini frame:

  • Tạo và hiển thị cửa sổ với WebView2
  • Quản lý WebView2 view
  • Tự động resize WebView khi cửa sổ thay đổi kích thước
  • Hỗ trợ URL ban đầu khi tạo cửa sổ
