Mô tả chi tiết

Dialog quản lý hợp đồng khám sức khỏe.

Class này kế thừa từ CGuiDialog, cung cấp các chức năng:

  • Tạo và chỉnh sửa hợp đồng khám sức khỏe
  • Quản lý thông tin hợp đồng: số hợp đồng, ngày ký, công ty, ngày khám
  • Chọn đối tượng và loại hợp đồng
  • Hỗ trợ miễn phí ăn sáng
  • Thêm công ty mới từ dialog
  • Lưu và cập nhật thông tin hợp đồng
