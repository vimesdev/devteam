Mô tả chi tiết

Dialog quản lý phiếu trả thuốc/vật tư của khoa.

Class này kế thừa từ CGuiDialog, cung cấp các chức năng:

  • Quản lý danh sách và chi tiết phiếu trả thuốc/vật tư
  • Sử dụng tab để chuyển đổi giữa danh sách và chi tiết
  • Tích hợp CWardReturnOrderList và CWardReturnOrder
