Mô tả chi tiết

Dialog đổi mật khẩu người dùng.

Class này kế thừa từ CGuiDialog, cung cấp các chức năng:

  • Đổi mật khẩu cho người dùng hệ thống
  • Xác thực mật khẩu cũ
  • Nhập và xác nhận mật khẩu mới
  • Kiểm tra tính hợp lệ của mật khẩu mới
  • Áp dụng thay đổi mật khẩu
