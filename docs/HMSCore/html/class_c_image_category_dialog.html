Mô tả chi tiết

Dialog quản lý danh mục hình ảnh.

Class này kế thừa từ CGuiDialog, cung cấp các chức năng:

  • Hiển thị danh sách các danh mục hình ảnh
  • Thêm, sửa, xóa danh mục hình ảnh
  • Xem và chỉnh sửa hình ảnh
  • Áp dụng thay đổi cho danh mục
  • Quản lý file hình ảnh và metadata
