Mô tả chi tiết

View phân khám chuyển khoa dạng HTML.

Class này kế thừa từ CGuiView, cung cấp các chức năng:

  • Hiển thị và chỉnh sửa phân khám chuyển khoa dạng HTML
  • Chọn loại khám
  • Load và lưu dữ liệu dạng JSON
  • In phân khám chuyển khoa
  • Quản lý form HTML động
