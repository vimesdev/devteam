Mô tả chi tiết

Grid control hỗ trợ nhập liệu trực tiếp với nhiều tùy chọn.

Class này kế thừa từ CMFCGridCtrl, cung cấp các chức năng:

  • Nhập liệu trực tiếp trên grid
  • Nhiều tùy chọn điều khiển hành vi edit: ENTER, TAB, arrow keys
  • Tự động thêm dòng mới khi nhấn ENTER/DOWN ở dòng cuối
  • Quản lý in-place editing
  • Thêm và chèn record mới
