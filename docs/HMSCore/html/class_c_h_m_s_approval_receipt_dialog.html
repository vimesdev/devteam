Mô tả chi tiết

Dialog duyệt và in phiếu thu.

Class này kế thừa từ CGuiDialog, cung cấp các chức năng:

  • Duyệt và quản lý phiếu thu
  • Tính toán tổng tiền, tiền đặt cọc, tiền phải trả
  • Quản lý thanh toán và hoàn tiền
  • In phiếu thu cho nhiều loại dịch vụ (PACS, Hospital Fee, PM)
  • Quản lý danh sách hóa đơn
  • Tùy chọn không tạo đơn đã thanh toán
