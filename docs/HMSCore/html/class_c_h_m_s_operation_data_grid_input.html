Mô tả chi tiết

Dialog nhập dữ liệu phẫu thuật dạng grid động.

Class này kế thừa từ CGuiDialog, cung cấp các chức năng:

  • Nhập dữ liệu phẫu thuật dạng grid động
  • Khởi tạo cột và header từ form definition
  • Hỗ trợ nhiều loại control: text, combo
  • Thêm, xóa dòng dữ liệu
  • Xác nhận và lưu dữ liệu
  • Tự động scroll và quản lý focus
