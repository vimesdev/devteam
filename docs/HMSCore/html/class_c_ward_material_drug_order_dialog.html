Mô tả chi tiết

Dialog quản lý đơn hàng vật tư/thuốc của khoa.

Class này kế thừa từ CGuiDialog, cung cấp các chức năng:

  • Quản lý danh sách và chi tiết đơn hàng vật tư/thuốc
  • Sử dụng tab để chuyển đổi giữa danh sách và chi tiết
  • Tích hợp CWardMaterialDrugOrderList và CWardMaterialDrugOrder
