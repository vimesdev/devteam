Mô tả chi tiết

Dialog thiết lập template cho form HTML.

Class này kế thừa từ CGuiDialog, cung cấp các chức năng:

  • Thiết lập và chỉnh sửa template cho các form HTML
  • Quản lý danh sách các item trong template
  • Sử dụng WebView2 để preview template
  • Áp dụng các thiết lập cho template
  • Quản lý mã SNOMED và các mã chuẩn khác
  • Cập nhật cell trong grid
