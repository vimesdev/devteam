Mô tả chi tiết

Dialog quản lý danh sách lựa chọn (selection list).

Class này kế thừa từ CGuiDialog, cung cấp dialog generic:

  • Quản lý danh sách các item với ID và Description
  • Thêm, sửa, xóa các item trong danh sách
  • Lưu và cập nhật thông tin
  • Hỗ trợ nhiều loại selection list khác nhau thông qua tham số Type
  • Generic dialog có thể tái sử dụng cho nhiều mục đích
