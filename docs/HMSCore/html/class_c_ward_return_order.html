Mô tả chi tiết

View quản lý chi tiết phiếu trả thuốc/vật tư của khoa.

Class này kế thừa từ CGuiView, cung cấp các chức năng:

  • Quản lý thông tin chung của phiếu trả (số phiếu, ngày, kho, phòng ban)
  • Quản lý danh sách các dòng phiếu trả
  • Thêm, sửa, xóa các dòng phiếu trả
  • Duyệt và in phiếu trả
  • Tính toán tổng số dòng và tổng tiền
  • Quản lý trạng thái phiếu trả
