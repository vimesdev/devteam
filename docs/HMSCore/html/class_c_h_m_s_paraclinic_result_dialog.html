Mô tả chi tiết

Dialog nhập kết quả cận lâm sàng.

Class này kế thừa từ CGuiDialog, cung cấp các chức năng:

  • Nhập kết quả cận lâm sàng cho bệnh nhân
  • Lọc theo khoảng thời gian và khoa
  • Chọn item cận lâm sàng và nhập mô tả kết quả
  • Đánh dấu tất cả hoặc bỏ đánh dấu
  • Áp dụng kết quả cho nhiều đơn cùng lúc
  • Lưu tổng hợp kết quả cận lâm sàng
