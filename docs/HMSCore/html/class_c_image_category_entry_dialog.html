Mô tả chi tiết

Dialog nhập danh mục hình ảnh.

Class này kế thừa từ CGuiDialog, cung cấp các chức năng:

  • Thêm và chỉnh sửa danh mục hình ảnh
  • Quản lý cấu trúc phân cấp của danh mục (parent-child)
  • Nhập mã, tên và mô tả cho danh mục
  • Chọn loại và danh mục cha
  • Lưu và cập nhật thông tin danh mục
