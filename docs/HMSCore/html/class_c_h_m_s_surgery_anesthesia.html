Mô tả chi tiết

View quản lý theo dõi gây mê trong phẫu thuật.

Class này kế thừa từ CGuiView, cung cấp các chức năng:

  • Quản lý danh sách các lần theo dõi gây mê
  • Thêm, sửa, xóa thông tin theo dõi gây mê
  • Hiển thị biểu đồ theo dõi gây mê
  • Load dữ liệu theo Order ID
  • Quản lý scale cho biểu đồ
