Mô tả chi tiết

View quản lý hồ sơ phẫu thuật.

Class này kế thừa từ CGuiView, cung cấp các chức năng:

  • Quản lý danh sách hồ sơ phẫu thuật
  • Thêm, sửa, xóa hồ sơ phẫu thuật
  • Ký số hồ sơ phẫu thuật
  • Load dữ liệu theo Order ID
  • Quản lý scale cho hiển thị
