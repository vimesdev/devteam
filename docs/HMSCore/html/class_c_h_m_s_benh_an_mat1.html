Mô tả chi tiết

View hiển thị bệnh án mẫu 1 (form HTML).

Class này kế thừa từ CGuiView, cung cấp các chức năng:

  • Hiển thị và chỉnh sửa bệnh án mẫu 1 dạng HTML
  • Load thông tin cơ bản, khoa phòng, chẩn đoán
  • Load vital signs và thông tin điều trị
  • Merge JSON data từ nhiều nguồn
  • In bệnh án
  • Hỗ trợ cả HTMLayout và WebView2
  • Thay đổi layout theo mode
