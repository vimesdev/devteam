Mô tả chi tiết

View quản lý đơn cận lâm sàng (xét nghiệm, chẩn đoán hình ảnh, etc.).

Class này kế thừa từ CGuiView, cung cấp các chức năng:

  • Tạo và quản lý đơn cận lâm sàng
  • Thêm nhóm và item vào đơn
  • Xác nhận và hủy đơn
  • Nhập kết quả cận lâm sàng
  • Xem hình ảnh DICOM
  • In đơn và kết quả
  • Phân công giải phẫu bệnh
  • Lọc theo trạng thái ký số
  • Quản lý nhiều loại cận lâm sàng: xét nghiệm, X-quang, siêu âm, etc.
