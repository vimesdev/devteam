Mô tả chi tiết

Lớp mở rộng của CHTMLayoutCtrl với các tính năng bổ sung cho EMR forms.

Class này kế thừa từ CHTMLayoutCtrl và cung cấp các chức năng mở rộng:

  • Tính toán các chỉ số y tế: BMI, BSA, điểm dinh dưỡng
  • Xử lý các form điều trị đặc biệt: trượt ngã, hoạt trị liệu, dinh dưỡng, etc.
  • Setting mode để chỉnh sửa form
  • Callback events cho các hành động của user
  • Load dữ liệu vital signs và thông tin điều trị
  • Xử lý menu context và mouse events
  • Tính toán điểm số cho các form đánh giá
