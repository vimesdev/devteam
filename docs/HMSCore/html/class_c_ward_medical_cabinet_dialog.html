Mô tả chi tiết

Dialog quản lý tủ thuốc của khoa.

Class này kế thừa từ CGuiDialog, cung cấp các chức năng:

  • Quản lý danh sách và chi tiết đơn hàng tủ thuốc
  • Quản lý danh sách và chi tiết công nợ tủ thuốc
  • Sử dụng tab để chuyển đổi giữa các view
  • Tích hợp CWardMedicalCabinetOrderList, CWardMedicalCabinetOrder và CWardMedicalDebtView
  • Hỗ trợ hoàn trả thuốc tự trực
