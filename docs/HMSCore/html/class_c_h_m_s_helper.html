Mô tả chi tiết

Lớp helper cung cấp các hàm tiện ích cho ứng dụng HMS.

Class này cung cấp các hàm tiện ích:

  • Chuẩn bị SQL cho phẫu thuật lặp lại
  • Kiểm tra mã đối tượng KCB và mã loại KCB
  • Validate thông tin bảo hiểm
  • Xử lý thông tin từ card reader
  • Parse và decode base64 image
  • Kiểm tra tương tác thuốc
  • Kiểm tra lịch khám dinh dưỡng
  • Convert record từ database sang JSON
  • Set JSON value cho HTMLayout control
