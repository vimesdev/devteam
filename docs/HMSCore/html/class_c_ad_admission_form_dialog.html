Mô tả chi tiết

Dialog quản lý form nhập viện.

Class này kế thừa từ CGuiDialog, cung cấp các chức năng:

  • Quản lý form nhập viện với ID cũ và tên mới
  • Đổi tên form nhập viện
  • Thêm mô tả cho form
  • Quản lý trạng thái active/inactive của form
  • Lưu và cập nhật thông tin form
