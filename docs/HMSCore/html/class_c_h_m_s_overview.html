Mô tả chi tiết

View tổng quan điều trị với nhiều tab template.

Class này kế thừa từ CGuiView, cung cấp các chức năng:

  • Hiển thị tổng quan điều trị bằng HTMLayout
  • Quản lý nhiều tab template: tổng quan, ngã, dinh dưỡng, bàn giường
  • Load và refresh dữ liệu điều trị
  • Quản lý các template HTML động
