Mô tả chi tiết

View chính quản lý bệnh án điện tử với nhiều tab.

Class này kế thừa từ CGuiView, cung cấp view chính:

  • Quản lý nhiều tab: Patient Profile, Patient Information, Signature View
  • Tích hợp CEMRPatientProfile, CHMSPatientInformation, CHMSSignatureView
  • Load document theo patient number hoặc document number
  • Hỗ trợ cả nội trú và ngoại trú
  • Chuyển đổi giữa các tab view
