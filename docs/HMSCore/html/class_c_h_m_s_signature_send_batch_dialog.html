Mô tả chi tiết

Dialog gửi hàng loạt tài liệu để ký số

Class này kế thừa từ CGuiDialog, cung cấp các chức năng:

  • Gửi hàng loạt tài liệu để ký số
  • Chọn khoa, người gửi, người nhận, và trường ký
  • Quản lý danh sách tài liệu cần gửi
  • Áp dụng các thiết lập ký số cho nhiều tài liệu cùng lúc
  • Quản lý trạng thái đã đăng (posted)
  • Lọc theo khoảng thời gian, loại tài liệu, và form ID
