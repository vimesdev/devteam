Mô tả chi tiết

Dialog quản lý việc trả thuốc của bệnh nhân tại khoa.

Class này kế thừa từ CGuiDialog, cung cấp các chức năng:

  • Hiển thị danh sách bệnh nhân và danh sách thuốc cần trả
  • Xử lý việc trả thuốc từ khoa về kho
  • Preview và apply các giao dịch trả thuốc
  • Quản lý transaction line cho việc trả thuốc
  • Hỗ trợ nhiều loại giao dịch (EM Operation, Out Patient, etc.)
