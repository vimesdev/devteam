Mô tả chi tiết

Popup dialog nhập đơn thuốc ung thư

Class này kế thừa từ CGuiPopup, cung cấp các chức năng:

  • Nhập đơn thuốc ung thư trong popup window
  • Chọn tổ chức, kho, thuốc, đơn vị, số lượng
  • Quản lý liều dùng, cách dùng và tốc độ truyền
  • Hỗ trợ thuốc ngoại và tìm kiếm generic
  • Bao gồm kho dịch vụ
