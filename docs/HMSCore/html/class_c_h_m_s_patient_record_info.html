Mô tả chi tiết

View quản lý thông tin chi tiết hồ sơ bệnh nhân và nhật ký mượn/trả

Class này kế thừa từ CGuiView, cung cấp các chức năng:

  • Quản lý thông tin chi tiết hồ sơ bệnh nhân: thông tin cá nhân, nhập viện, xuất viện
  • Quản lý vị trí lưu trữ: phòng, tủ, ngăn kéo, vị trí
  • Quản lý nhật ký mượn/trả hồ sơ
  • Tìm kiếm hồ sơ theo nhiều tiêu chí
  • In hồ sơ và tổng hợp
  • Quản lý trạng thái hồ sơ và mượn/trả
  • Hỗ trợ nhiều mode: borrow, return, edit, view
