Mô tả chi tiết

Popup dialog nhập đơn thuốc điều trị (phiên bản 2).

Class này kế thừa từ CGuiPopup, cung cấp các chức năng:

  • Nhập đơn thuốc điều trị trong popup window
  • Chọn kho, thuốc, số lượng, chỉ định
  • Quản lý tốc độ truyền và cách dùng
  • Quản lý đơn thuốc ngoại và kho bảo hiểm
  • Quản lý thời gian và số lần dùng
  • Hỗ trợ tìm kiếm generic
