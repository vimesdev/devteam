Mô tả chi tiết

Dialog mới để nhập và quản lý thông tin điều trị với HTML form.

Class này kế thừa từ CGuiDialog, cung cấp dialog điều trị:

  • Hiển thị và chỉnh sửa form điều trị dạng HTML
  • Quản lý danh sách các form điều trị
  • Load thông tin phẫu thuật, chẩn đoán, khoa phòng
  • Tạo barcode cho bệnh nhân
  • Switch giữa HTML view và WebView2
  • Thêm, sửa, xóa, lưu form điều trị
  • Print và export form
  • Quản lý multiple views và tabs
