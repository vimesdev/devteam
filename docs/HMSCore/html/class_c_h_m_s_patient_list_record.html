Mô tả chi tiết

View danh sách bệnh nhân có hồ sơ lưu trữ

Class này kế thừa từ CGuiView, cung cấp các chức năng:

  • Tìm kiếm bệnh nhân theo khoảng thời gian, số bệnh án, khoa, tên
  • Quản lý số lưu trữ và tổng số hồ sơ
  • Lọc theo loại điều trị
  • Đánh dấu bệnh nhân đã tử vong
  • Export danh sách ra file
  • Hỗ trợ cả nội trú và ngoại trú
