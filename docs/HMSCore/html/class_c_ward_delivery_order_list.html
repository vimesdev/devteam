Mô tả chi tiết

View hiển thị danh sách phiếu giao hàng của khoa.

Class này kế thừa từ CGuiView, cung cấp các chức năng:

  • Hiển thị danh sách các phiếu giao hàng
  • Tìm kiếm và lọc theo khoảng thời gian, loại giao dịch, kho, phòng ban
  • Quản lý các trạng thái: tất cả, mở, đã duyệt, đang gửi
  • Xem chi tiết phiếu giao hàng
  • Tích hợp CWardDeliveryOrder để hiển thị chi tiết
