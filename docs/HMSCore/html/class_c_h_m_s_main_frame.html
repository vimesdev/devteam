Mô tả chi tiết

Lớp main frame chính của ứng dụng HMS (Hospital Management System).

Class này kế thừa từ CGuiMainFrame, là lớp chính quản lý toàn bộ ứng dụng HMS:

  • Quản lý thông tin bệnh nhân, hồ sơ, và các phiếu khám
  • Xử lý các giao dịch với cơ sở dữ liệu
  • Quản lý các module: Reception, Treatment, Pharmacy, Operation, etc.
  • In các loại phiếu: đơn thuốc, phiếu lĩnh thuốc, phiếu khám, phiếu xuất viện, etc.
  • Tích hợp với các dịch vụ: BHYT, POS, Smart Card, PACS, DICOM
  • Quản lý user, department, và permissions
  • Xử lý các giao dịch tài chính và thanh toán
  • Hỗ trợ ký số điện tử và digital signature
  • Tích hợp với các dịch vụ web và API
