Mô tả chi tiết

Dialog quản lý đơn thuốc ung thư với nhiều thuốc.

Class này kế thừa từ CGuiDialog, cung cấp các chức năng:

  • Quản lý đơn thuốc ung thư với nhiều thuốc (tối đa 3 thuốc)
  • Chọn kho, số lượng đặt, thuốc và số lượng cho từng thuốc
  • Quản lý cách dùng và mô tả cách dùng cho từng thuốc
  • Quản lý tên phác đồ, đơn vị, liều dùng
  • Bật/tắt thuốc thứ 2 và thứ 3
  • Lưu và cập nhật đơn thuốc ung thư
