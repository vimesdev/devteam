Mô tả chi tiết

View hiển thị danh sách phiếu trả thuốc/vật tư của khoa.

Class này kế thừa từ CGuiView, cung cấp các chức năng:

  • Hiển thị danh sách các phiếu trả thuốc/vật tư
  • Tìm kiếm và lọc theo khoảng thời gian, trạng thái, kho
  • Lọc theo loại giao dịch
  • Xem chi tiết phiếu trả
  • Quản lý các trạng thái: tất cả, mở, đã duyệt, đang gửi
