Mô tả chi tiết

View quản lý đơn phẫu thuật.

Class này kế thừa từ CGuiView, cung cấp các chức năng:

  • Tạo và quản lý đơn phẫu thuật
  • Quản lý thông tin chung, vật liệu, và sequence phẫu thuật
  • Xem hình ảnh DICOM
  • In đơn phẫu thuật và các tài liệu liên quan
  • Cập nhật thông tin phẫu thuật
  • Theo dõi điều trị laser
  • Quản lý nhiều tab: General, Material, IN Material, Preparing Material, Profiles, Sequence
