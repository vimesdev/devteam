Mô tả chi tiết

Dialog quản lý đơn bán hàng cho bệnh nhân.

Class này kế thừa từ CGuiDialog, cung cấp các chức năng:

  • Tạo và quản lý đơn bán hàng cho bệnh nhân
  • Quản lý thông tin khách hàng và đơn hàng
  • Thêm, sửa, xóa các dòng trong đơn hàng
  • Quản lý chỉ định, số ngày, số lần dùng thuốc
  • Xác nhận và duyệt đơn hàng
  • In phiếu giao hàng thuốc
  • Hỗ trợ thực phẩm chức năng
  • Tạo template đơn thuốc
  • Quản lý nguồn thanh toán
