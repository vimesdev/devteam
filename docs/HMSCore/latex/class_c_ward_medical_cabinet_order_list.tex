View hiển thị danh sách đơn hàng tủ thuốc của khoa.

\begin{DoxyItemize}

\item Tìm kiếm và lọc theo khoảng thời gian, trạng thái, kho
\item Lọc theo loại giao dịch
\item Lọc theo đơn hàng có dòng hay không
\item Xem chi tiết đơn hàng tủ thuốc
\item Quản lý các trạng thái\+: tất cả, mở, đã duyệt, đang gửi
