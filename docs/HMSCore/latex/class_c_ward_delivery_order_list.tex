\doxysubsection{Mô tả chi tiết}
View hiển thị danh sách phiếu giao hàng của khoa. 

Class này kế thừa từ CGui\+View, cung cấp các chức năng\+:
\begin{DoxyItemize}
\item Hiển thị danh sách các phiếu giao hàng
\item Tìm kiếm và lọc theo khoảng thời gian, loại giao dịch, kho, phòng ban
\item Quản lý các trạng thái\+: tất cả, mở, đã duyệt, đang gửi
\item Xem chi tiết phiếu giao hàng
\item Tích hợp \doxylink{class_c_ward_delivery_order}{CWard\+Delivery\+Order} để hiển thị chi tiết 
\end{DoxyItemize}
