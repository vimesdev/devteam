\doxysubsection{Mô tả chi tiết}
Dialog copy đơn thuốc từ lần điều trị trước. 

Class này kế thừa từ CGui\+Dialog, cung cấp các chức năng\+:
\begin{DoxyItemize}
\item Copy đơn thuốc từ các lần điều trị trước
\item Hiển thị danh sách các đơn thuốc theo khoảng thời gian
\item Chọn đơn thuốc cần copy
\item Áp dụng đơn thuốc đã chọn cho điều trị hiện tại
\item Lọc theo khoảng thời gian (period) 
\end{DoxyItemize}
