\doxysubsection{Mô tả chi tiết}
Dialog nhập dữ liệu phẫu thuật dạng grid động. 

Class này kế thừa từ CGui\+Dialog, cung cấp các chức năng\+:
\begin{DoxyItemize}
\item Nhập dữ liệu phẫu thuật dạng grid động
\item Khởi tạo cột và header từ form definition
\item Hỗ trợ nhiều loại control\+: text, combo
\item Thêm, xóa dòng dữ liệu
\item Xác nhận và lưu dữ liệu
\item Tự động scroll và quản lý focus 
\end{DoxyItemize}
