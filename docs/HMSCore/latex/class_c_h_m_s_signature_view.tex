\doxysubsection{Mô tả chi tiết}
View quản lý chữ ký số cho các tài liệu. 

Class này kế thừa từ CGui\+View, cung cấp các chức năng\+:
\begin{DoxyItemize}
\item Hiển thị danh sách tài liệu cần ký số
\item Lọc theo danh mục và trạng thái ký số (tất cả, chưa ký, đã ký, hoàn thành)
\item Thêm và xóa tài liệu
\item Gửi tài liệu để ký số
\item Hiển thị tổng hợp số lượng tài liệu theo trạng thái
\item Hỗ trợ nhiều loại view\+: bác sĩ, trưởng khoa, giám đốc
\item Tìm kiếm tài liệu theo từ khóa 
\end{DoxyItemize}
