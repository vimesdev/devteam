\doxysubsection{Mô tả chi tiết}
Dialog để nhập và quản lý thông tin điều trị với HTML form. 

Class này kế thừa từ CGui\+Dialog, cung cấp dialog điều trị\+:
\begin{DoxyItemize}
\item Hiển thị và chỉnh sửa form điều trị dạng HTML
\item Quản lý danh sách các form điều trị
\item Load thông tin phẫu thuật, chẩn đoán, khoa phòng, hồ sơ lâm sàng
\item Tạo barcode cho bệnh nhân
\item Thêm, sửa, xóa, lưu form điều trị
\item Print và export form
\item Hỗ trợ single view và multiple views mode
\item Update mode để chỉnh sửa form đã có 
\end{DoxyItemize}
