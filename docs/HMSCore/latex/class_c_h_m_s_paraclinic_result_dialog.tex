\doxysubsection{Mô tả chi tiết}
Dialog nhập kết quả cận lâm sàng. 

Class này kế thừa từ CGui\+Dialog, cung cấp các chức năng\+:
\begin{DoxyItemize}
\item Nhập kết quả cận lâm sàng cho bệnh nhân
\item Lọc theo khoảng thời gian và khoa
\item Chọn item cận lâm sàng và nhập mô tả kết quả
\item Đánh dấu tất cả hoặc bỏ đánh dấu
\item Áp dụng kết quả cho nhiều đơn cùng lúc
\item Lưu tổng hợp kết quả cận lâm sàng
