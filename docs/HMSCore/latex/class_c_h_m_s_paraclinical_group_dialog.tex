Dialog quản lý nhóm cận lâm sàng. 

Class này kế thừa từ CGui\+Dialog, cung cấp các chức năng\+:
\begin{DoxyItemize}
\item Quản lý các nhóm cận lâm sàng
\item Chọn và áp dụng nhóm cận lâm sàng cho bệnh nhân
\item Quản lý danh sách item trong nhóm
\item Tự động in và ký số đơn
\item Quản lý ngày đặt
\item Kiểm tra cảnh báo phí và đặt cọc
