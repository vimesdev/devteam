\doxysubsection{Mô tả chi tiết}
Lớp client để liên thông với hệ thống Dược Quốc Gia. 

Class này kế thừa từ CHttp\+Client, cung cấp các chức năng\+:
\begin{DoxyItemize}
\item Kết nối và xác thực với hệ thống Dược Quốc Gia
\item Lấy token và quản lý session
\item Nhận lịch sử khám chữa bệnh và chi tiết KCB
\item Kiểm tra thông tin khám chữa bệnh và trạng thái hồ sơ
\item Thêm hồ sơ bệnh án và thông tin vào khám
\item Gửi đơn thuốc, hóa đơn thuốc, phiếu nhập/xuất kho
\item Xóa đơn thuốc và hóa đơn thuốc
\item Tạo mới danh mục thuốc QG
\item Tự động gửi purchase order và sale order 
\end{DoxyItemize}
