\doxysubsection{Mô tả chi tiết}
Dialog chọn danh sách từ một danh sách có sẵn. 

Class này kế thừa từ CGui\+Dialog, cung cấp dialog lựa chọn\+:
\begin{DoxyItemize}
\item Hiển thị danh sách các item để chọn
\item Chọn nhiều item từ danh sách
\item Quản lý danh sách đã chọn riêng biệt
\item Chọn tất cả hoặc bỏ chọn tất cả
\item Lấy thông tin item đã chọn theo index
\item Hỗ trợ checkbox để enable/disable chức năng chọn
\item Load dữ liệu từ SQL query hoặc JSON 
\end{DoxyItemize}
