\doxysubsection{Mô tả chi tiết}
Class quản lý đọc thẻ \doxylink{class_a_t_m}{ATM} sử dụng PC/\+SC (Smart Card) API. 

Class này cung cấp các chức năng để\+:
\begin{DoxyItemize}
\item Kết nối và quản lý smart card reader
\item Đọc và ghi dữ liệu từ thẻ \doxylink{class_a_t_m}{ATM}
\item Xác thực thẻ và load key
\item Quản lý trạng thái thẻ và reader
\item Hỗ trợ timer để theo dõi trạng thái thẻ 
\end{DoxyItemize}
