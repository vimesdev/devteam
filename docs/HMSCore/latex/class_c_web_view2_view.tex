\doxysubsection{Mô tả chi tiết}
Lớp view tích hợp Microsoft Web\+View2 để hiển thị web content trong ứng dụng. 

Class này kế thừa từ CView, cung cấp Web\+View2 integration\+:
\begin{DoxyItemize}
\item Hiển thị web content sử dụng Microsoft Edge Web\+View2
\item Navigate đến URL hoặc HTML content
\item Execute Java\+Script và nhận kết quả
\item Gửi/nhận messages giữa ứng dụng và web content
\item Hỗ trợ navigation events và process failure handling
\item Inject script vào web page
\item Login và authentication
\item Clear cache và quản lý cookies
\item High DPI awareness
\item Static methods để khởi tạo environment và utility functions 
\end{DoxyItemize}
