\doxysubsection{Mô tả chi tiết}
Dialog preview và in các tài liệu của bệnh nhân. 

Class này kế thừa từ CGui\+Dialog, cung cấp các chức năng\+:
\begin{DoxyItemize}
\item Hiển thị danh sách các tài liệu của bệnh nhân
\item Preview và in phiếu khám
\item Xem theo số CMND/\+CCCD
\item Xem tài liệu đã chọn hoặc tất cả
\item In nhiều phiếu khám cùng lúc
\item Quản lý danh sách document numbers 
\end{DoxyItemize}
