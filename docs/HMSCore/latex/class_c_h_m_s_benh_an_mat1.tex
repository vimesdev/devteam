View hiển thị bệnh án mẫu 1 (form HTML). 

Class này kế thừa từ CGui\+View, cung cấp các chức năng\+:
\begin{DoxyItemize}
\item Hiển thị và chỉnh sửa bệnh án mẫu 1 dạng HTML

\item Load vital signs và thông tin điều trị
\item Merge JSON data từ nhiều nguồn
