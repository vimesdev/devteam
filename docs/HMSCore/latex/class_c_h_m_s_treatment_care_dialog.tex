\doxysubsection{Mô tả chi tiết}
Dialog quản lý phiếu chăm sóc bệnh nhân. 

Class này kế thừa từ CGui\+Dialog, cung cấp các chức năng\+:
\begin{DoxyItemize}
\item Quản lý phiếu chăm sóc bệnh nhân theo tháng
\item Thêm, sửa, xóa các phiếu chăm sóc
\item Quản lý phiếu chăm sóc trước phẫu thuật và sau phẫu thuật
\item Quản lý phiếu chăm sóc của điều dưỡng
\item In phiếu chăm sóc và theo dõi nhiệt độ huyết áp
\item Khóa và mở khóa phiếu chăm sóc
\item Copy phiếu chăm sóc
\item Quản lý theo dõi công khai 
\end{DoxyItemize}
