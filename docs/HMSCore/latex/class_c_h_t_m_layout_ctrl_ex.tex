\doxysubsection{Mô tả chi tiết}
Lớp mở rộng của \doxylink{class_c_h_t_m_layout_ctrl}{CHTMLayout\+Ctrl} với các tính năng bổ sung cho EMR forms. 

Class này kế thừa từ \doxylink{class_c_h_t_m_layout_ctrl}{CHTMLayout\+Ctrl} và cung cấp các chức năng mở rộng\+:
\begin{DoxyItemize}
\item Tính toán các chỉ số y tế\+: BMI, BSA, điểm dinh dưỡng
\item Xử lý các form điều trị đặc biệt\+: trượt ngã, hoạt trị liệu, dinh dưỡng, etc.
\item Setting mode để chỉnh sửa form
\item Callback events cho các hành động của user
\item Load dữ liệu vital signs và thông tin điều trị
\item Xử lý menu context và mouse events
\item Tính toán điểm số cho các form đánh giá 
\end{DoxyItemize}
