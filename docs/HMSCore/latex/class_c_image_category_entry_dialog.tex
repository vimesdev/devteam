\doxysubsection{Mô tả chi tiết}
Dialog nhập danh mục hình ảnh. 

Class này kế thừa từ CGui\+Dialog, cung cấp các chức năng\+:
\begin{DoxyItemize}
\item Thêm và chỉnh sửa danh mục hình ảnh
\item Quản lý cấu trúc phân cấp của danh mục (parent-\/child)
\item Nhập mã, tên và mô tả cho danh mục
\item Chọn loại và danh mục cha
\item Lưu và cập nhật thông tin danh mục 
\end{DoxyItemize}
