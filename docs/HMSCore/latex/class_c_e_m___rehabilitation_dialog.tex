Dialog quản lý phục hồi chức năng. 

Class này kế thừa từ CGui\+Dialog, cung cấp các chức năng\+:

\item Quản lý phương pháp điều trị, máy móc, liều lượng kỹ thuật
\item Quản lý số lần điều trị và thời gian điều trị
\item Quản lý sổ và tờ trong sổ phục hồi chức năng
\item Quản lý bệnh chính và bệnh kèm theo
\item In phiếu phục hồi chức năng
\item Quản lý thông tin mở rộng dạng JSON
