\doxysubsection{Mô tả chi tiết}
Lớp quản lý các tùy chọn cấu hình của ứng dụng. 

Class này cung cấp cách lưu trữ và truy xuất các tùy chọn\+:
\begin{DoxyItemize}
\item Lưu trữ các option dạng key-\/value (wide string)
\item Lấy giá trị option theo tên
\item Sử dụng map để lưu trữ hiệu quả
\item Global instance the\+Options để truy cập từ mọi nơi 
\end{DoxyItemize}
