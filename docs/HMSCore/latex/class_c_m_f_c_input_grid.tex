\doxysubsection{Mô tả chi tiết}
Grid control hỗ trợ nhập liệu trực tiếp với nhiều tùy chọn. 

Class này kế thừa từ CMFCGrid\+Ctrl, cung cấp các chức năng\+:
\begin{DoxyItemize}
\item Nhập liệu trực tiếp trên grid
\item Nhiều tùy chọn điều khiển hành vi edit\+: ENTER, TAB, arrow keys
\item Tự động thêm dòng mới khi nhấn ENTER/\+DOWN ở dòng cuối
\item Quản lý in-\/place editing
\item Thêm và chèn record mới 
\end{DoxyItemize}
