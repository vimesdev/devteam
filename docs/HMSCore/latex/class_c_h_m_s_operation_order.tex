\doxysubsection{Mô tả chi tiết}
View quản lý đơn phẫu thuật. 

Class này kế thừa từ CGui\+View, cung cấp các chức năng\+:
\begin{DoxyItemize}
\item Tạo và quản lý đơn phẫu thuật
\item Quản lý thông tin chung, vật liệu, và sequence phẫu thuật
\item Xem hình ảnh DICOM
\item In đơn phẫu thuật và các tài liệu liên quan
\item Cập nhật thông tin phẫu thuật
\item Theo dõi điều trị laser
\item Quản lý nhiều tab\+: General, Material, IN Material, Preparing Material, Profiles, Sequence 
\end{DoxyItemize}
