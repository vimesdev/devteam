\doxysubsection{Mô tả chi tiết}
Lớp mini frame chứa Web\+View2 để hiển thị web content trong cửa sổ riêng. 

Class này kế thừa từ CGui\+Mini\+Frame, cung cấp cửa sổ mini frame\+:
\begin{DoxyItemize}
\item Tạo và hiển thị cửa sổ với Web\+View2
\item Quản lý Web\+View2 view
\item Tự động resize Web\+View khi cửa sổ thay đổi kích thước
\item Hỗ trợ URL ban đầu khi tạo cửa sổ 
\end{DoxyItemize}
