\doxysubsection{Mô tả chi tiết}
Dialog thiết lập các item trong bệnh án điện tử 

Class này kế thừa từ CGui\+Dialog, cung cấp các chức năng\+:
\begin{DoxyItemize}
\item Quản lý các item trong bệnh án điện tử
\item Load và hiển thị các form HTML
\item Copy item từ form khác
\item Thêm, sửa, xóa các item
\item Lưu và cập nhật JSON data
\item Quản lý list các item với HTML layout control 
\end{DoxyItemize}
