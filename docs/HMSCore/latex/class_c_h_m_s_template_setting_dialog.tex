\doxysubsection{Mô tả chi tiết}
Dialog thiết lập template cho form HTML. 

Class này kế thừa từ CGui\+Dialog, cung cấp các chức năng\+:
\begin{DoxyItemize}
\item Thiết lập và chỉnh sửa template cho các form HTML
\item Quản lý danh sách các item trong template
\item Sử dụng Web\+View2 để preview template
\item Áp dụng các thiết lập cho template
\item Quản lý mã SNOMED và các mã chuẩn khác
\item Cập nhật cell trong grid 
\end{DoxyItemize}
