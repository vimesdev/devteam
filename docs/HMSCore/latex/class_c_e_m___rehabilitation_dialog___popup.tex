\doxysubsection{Mô tả chi tiết}
Dialog popup nhập thông tin phục hồi chức năng. 

Class này kế thừa từ CGui\+Dialog, cung cấp các chức năng\+:
\begin{DoxyItemize}
\item Nhập thông tin phục hồi chức năng trong popup dialog
\item Quản lý ngày, số lần, phương pháp điều trị, liều lượng
\item Quản lý thời gian và số lần điều trị
\item Nhập ghi chú và người thực hiện
\item Lưu và cập nhật thông tin phục hồi chức năng 
\end{DoxyItemize}
