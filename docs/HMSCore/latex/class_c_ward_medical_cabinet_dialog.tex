Dialog quản lý tủ thuốc của khoa. 

Class này kế thừa từ CGui\+Dialog, cung cấp các chức năng\+:
\begin{DoxyItemize}
\item Quản lý danh sách và chi tiết đơn hàng tủ thuốc
\item Quản lý danh sách và chi tiết công nợ tủ thuốc
\item Sử dụng tab để chuyển đổi giữa các view
\item Tích hợp \doxylink{class_c_ward_medical_cabinet_order_list}{CWard\+Medical\+Cabinet\+Order\+List}, \doxylink{class_c_ward_medical_cabinet_order}{CWard\+Medical\+Cabinet\+Order} và \doxylink{class_c_ward_medical_debt_view}{CWard\+Medical\+Debt\+View}
