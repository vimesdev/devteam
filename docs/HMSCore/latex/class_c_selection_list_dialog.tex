\doxysubsection{Mô tả chi tiết}
Dialog quản lý danh sách lựa chọn (selection list). 

Class này kế thừa từ CGui\+Dialog, cung cấp dialog generic\+:
\begin{DoxyItemize}
\item Quản lý danh sách các item với ID và Description
\item Thêm, sửa, xóa các item trong danh sách
\item Lưu và cập nhật thông tin
\item Hỗ trợ nhiều loại selection list khác nhau thông qua tham số Type
\item Generic dialog có thể tái sử dụng cho nhiều mục đích 
\end{DoxyItemize}
