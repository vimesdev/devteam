\doxysubsection{Mô tả chi tiết}
Dialog quản lý danh mục hình ảnh. 

Class này kế thừa từ CGui\+Dialog, cung cấp các chức năng\+:
\begin{DoxyItemize}
\item Hiển thị danh sách các danh mục hình ảnh
\item Thêm, sửa, xóa danh mục hình ảnh
\item Xem và chỉnh sửa hình ảnh
\item Áp dụng thay đổi cho danh mục
\item Quản lý file hình ảnh và metadata 
\end{DoxyItemize}
