Dialog quản lý phiếu trả thuốc/vật tư của khoa. 

Class này kế thừa từ CGui\+Dialog, cung cấp các chức năng\+:
\begin{DoxyItemize}
\item Quản lý danh sách và chi tiết phiếu trả thuốc/vật tư
\item Sử dụng tab để chuyển đổi giữa danh sách và chi tiết
\item Tích hợp \doxylink{class_c_ward_return_order_list}{CWard\+Return\+Order\+List} và \doxylink{class_c_ward_return_order}{CWard\+Return\+Order}
