\doxysubsection{Mô tả chi tiết}
Lớp helper cung cấp các hàm tiện ích cho ứng dụng HMS. 

Class này cung cấp các hàm tiện ích\+:
\begin{DoxyItemize}
\item Chuẩn bị SQL cho phẫu thuật lặp lại
\item Kiểm tra mã đối tượng KCB và mã loại KCB
\item Validate thông tin bảo hiểm
\item Xử lý thông tin từ card reader
\item Parse và decode base64 image
\item Kiểm tra tương tác thuốc
\item Kiểm tra lịch khám dinh dưỡng
\item Convert record từ database sang JSON
\item Set JSON value cho HTMLayout control 
\end{DoxyItemize}
