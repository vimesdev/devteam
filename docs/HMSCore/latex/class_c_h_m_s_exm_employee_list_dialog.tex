\begin{DoxyItemize}
\item Tìm kiếm nhân viên theo tên
\item Chọn nhiều nhân viên từ danh sách
\item Chọn tất cả hoặc bỏ chọn tất cả

\item Load danh sách nhân viên theo công ty
