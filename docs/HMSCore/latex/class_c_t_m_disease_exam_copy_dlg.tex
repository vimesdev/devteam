\doxysubsection{Mô tả chi tiết}
Dialog copy diễn biến bệnh và đơn thuốc từ lần điều trị trước. 

Class này kế thừa từ CGui\+Dialog, cung cấp các chức năng\+:
\begin{DoxyItemize}
\item Copy diễn biến bệnh và đơn thuốc từ các lần điều trị trước
\item Hiển thị danh sách các lần điều trị
\item Chọn copy từ lần điều trị cuối cùng hoặc lần điều trị cụ thể
\item Load danh sách thuốc từ đơn thuốc đã chọn
\item Áp dụng copy cho điều trị hiện tại 
\end{DoxyItemize}
