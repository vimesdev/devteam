\doxysubsection{Mô tả chi tiết}
Dialog quản lý đơn bán hàng (sale order). 

Class này kế thừa từ CGui\+Dialog, cung cấp các chức năng\+:
\begin{DoxyItemize}
\item Tạo và quản lý đơn bán hàng
\item Quản lý thông tin khách hàng và đơn hàng
\item Thêm, sửa, xóa các dòng trong đơn hàng
\item Xác nhận và duyệt đơn hàng
\item In phiếu giao hàng và phiếu GLS
\item Quản lý thanh toán và trạng thái đơn hàng
\item Hỗ trợ thực phẩm chức năng
\item Liên kết với hệ thống Dược Quốc Gia
