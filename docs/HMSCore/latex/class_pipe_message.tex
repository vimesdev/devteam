\doxysubsection{Mô tả chi tiết}
Lớp đại diện cho message được truyền qua named pipe. 

Class này quản lý các message được gửi/nhận qua named pipe\+:
\begin{DoxyItemize}
\item Lưu trữ message ID và status
\item Chứa dữ liệu dạng string
\item Hỗ trợ parse và convert sang string 
\end{DoxyItemize}
