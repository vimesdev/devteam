\doxysubsection{Mô tả chi tiết}
View quản lý chuyển giao và ký số điện tử cho các tài liệu EMR. 

Class này kế thừa từ CGui\+View, cung cấp các chức năng\+:
\begin{DoxyItemize}
\item Tìm kiếm và lọc các tài liệu cần ký số
\item Ký số đơn lẻ hoặc hàng loạt
\item Chuyển giao tài liệu để ký số
\item Preview EMR trước khi ký
\item Tạo và gửi XML signature
\item Quản lý trạng thái ký số (mở, đã duyệt, hủy)
\item Lọc theo khoảng thời gian, loại tài liệu, khoa, số lượng chữ ký 
\end{DoxyItemize}
