Dialog chỉnh sửa thông tin phí của item. 

Class này kế thừa từ CGui\+Dialog, cung cấp các chức năng\+:
\begin{DoxyItemize}
\item Chỉnh sửa số lượng và phân bổ phí
\item Phân chia phí trong và ngoài phẫu thuật
\item Áp dụng đánh giá bảo hiểm
\item Quản lý số lượng thanh toán bảo hiểm và bệnh nhân
\item Hỗ trợ stent và các thiết bị y tế đặc biệt
\item Tùy chọn không tính trần
