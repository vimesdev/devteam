\doxysubsection{Mô tả chi tiết}
View danh sách bệnh nhân có hồ sơ lưu trữ 

Class này kế thừa từ CGui\+View, cung cấp các chức năng\+:
\begin{DoxyItemize}
\item Tìm kiếm bệnh nhân theo khoảng thời gian, số bệnh án, khoa, tên
\item Quản lý số lưu trữ và tổng số hồ sơ
\item Lọc theo loại điều trị
\item Đánh dấu bệnh nhân đã tử vong
\item Export danh sách ra file
\item Hỗ trợ cả nội trú và ngoại trú 
\end{DoxyItemize}
