\doxysubsection{Mô tả chi tiết}
Dialog quản lý tài liệu chờ ký số 

Class này kế thừa từ CGui\+Dialog, cung cấp các chức năng\+:
\begin{DoxyItemize}
\item Quản lý danh sách tài liệu chờ ký số
\item Chọn khoa, người gửi, người nhận, và trường ký
\item Áp dụng các thiết lập ký số
\item Quản lý trạng thái đã đăng (posted)
\item Lọc theo khoảng thời gian, loại tài liệu, và form ID 
\end{DoxyItemize}
