\doxysubsection{Mô tả chi tiết}
Dialog quản lý hợp đồng khám sức khỏe. 

Class này kế thừa từ CGui\+Dialog, cung cấp các chức năng\+:
\begin{DoxyItemize}
\item Tạo và chỉnh sửa hợp đồng khám sức khỏe
\item Quản lý thông tin hợp đồng\+: số hợp đồng, ngày ký, công ty, ngày khám
\item Chọn đối tượng và loại hợp đồng
\item Hỗ trợ miễn phí ăn sáng
\item Thêm công ty mới từ dialog
\item Lưu và cập nhật thông tin hợp đồng 
\end{DoxyItemize}
