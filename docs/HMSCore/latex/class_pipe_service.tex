\doxysubsection{Mô tả chi tiết}
Lớp dịch vụ quản lý named pipe để giao tiếp với các ứng dụng khác. 

Class này cung cấp dịch vụ named pipe\+:
\begin{DoxyItemize}
\item Tạo và quản lý named pipe server
\item Xử lý các kết nối từ client
\item Gửi/nhận messages qua pipe
\item Quản lý thread để xử lý các kết nối đồng thời
\item Hỗ trợ giao tiếp với các thiết bị ngoại vi như card reader, fingerprint scanner 
\end{DoxyItemize}
