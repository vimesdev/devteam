Popup dialog nhập đơn thuốc ung thư (phiên bản mở rộng). 

Class này kế thừa từ CGui\+Popup, cung cấp các chức năng\+:
\begin{DoxyItemize}
\item Nhập đơn thuốc ung thư trong popup window
\item Chọn thuốc, đơn vị, số lượng, chỉ định
\item Quản lý tốc độ truyền và cách dùng
\item Phiên bản mở rộng với các tính năng bổ sung cho điều trị ung thư
