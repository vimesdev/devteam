\doxysubsection{Mô tả chi tiết}
Dialog quản lý form nhập viện. 

Class này kế thừa từ CGui\+Dialog, cung cấp các chức năng\+:
\begin{DoxyItemize}
\item Quản lý form nhập viện với ID cũ và tên mới
\item Đổi tên form nhập viện
\item Thêm mô tả cho form
\item Quản lý trạng thái active/inactive của form
\item Lưu và cập nhật thông tin form 
\end{DoxyItemize}
