\doxysubsection{Mô tả chi tiết}
Dialog quản lý việc trả thuốc của bệnh nhân tại khoa. 

Class này kế thừa từ CGui\+Dialog, cung cấp các chức năng\+:
\begin{DoxyItemize}
\item Hiển thị danh sách bệnh nhân và danh sách thuốc cần trả
\item Xử lý việc trả thuốc từ khoa về kho
\item Preview và apply các giao dịch trả thuốc
\item Quản lý transaction line cho việc trả thuốc
\item Hỗ trợ nhiều loại giao dịch (EM Operation, Out Patient, etc.) 
\end{DoxyItemize}
