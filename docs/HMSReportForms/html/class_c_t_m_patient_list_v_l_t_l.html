Mô tả chi tiết

Lớp view báo cáo danh sách bệnh nhân vật lý trị liệu.

Class này kế thừa từ CGuiView, cung cấp báo cáo danh sách bệnh nhân vật lý trị liệu:

  • Lọc theo năm, kỳ báo cáo, khoảng thời gian
  • Lọc theo đối tượng, sổ, nhóm bệnh, khoa, phương pháp
  • Phân loại nội trú/ngoại trú
  • Hỗ trợ nhiều format báo cáo
  • Print preview và export Excel
  • Load dữ liệu từ database với các điều kiện lọc
