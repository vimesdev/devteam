Mô tả chi tiết

Lớp view báo cáo sổ phẫu thuật/thủ thuật.

Class này kế thừa từ CGuiView, cung cấp báo cáo sổ phẫu thuật/thủ thuật:

  • Lọc theo khoảng thời gian và đối tượng
  • Phân loại phẫu thuật hoặc thủ thuật
  • Export sang Excel với thông tin chi tiết
  • Bao gồm thông tin phẫu thuật và thuốc sử dụng
  • Load dữ liệu từ database và format Excel
