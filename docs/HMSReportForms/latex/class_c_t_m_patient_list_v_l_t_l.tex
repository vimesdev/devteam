\doxysubsection{Mô tả chi tiết}
Lớp view báo cáo danh sách bệnh nhân vật lý trị liệu. 

Class này kế thừa từ CGui\+View, cung cấp báo cáo danh sách bệnh nhân vật lý trị liệu\+:
\begin{DoxyItemize}
\item Lọc theo năm, kỳ báo cáo, khoảng thời gian
\item Lọc theo đối tượng, sổ, nhóm bệnh, khoa, phương pháp
\item Phân loại nội trú/ngoại trú
\item Hỗ trợ nhiều format báo cáo
\item Print preview và export Excel
\item Load dữ liệu từ database với các điều kiện lọc 
\end{DoxyItemize}
