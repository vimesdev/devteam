\doxysubsection{Mô tả chi tiết}
Lớp view báo cáo sổ phẫu thuật/thủ thuật. 

Class này kế thừa từ CGui\+View, cung cấp báo cáo sổ phẫu thuật/thủ thuật\+:
\begin{DoxyItemize}
\item Lọc theo khoảng thời gian và đối tượng
\item Phân loại phẫu thuật hoặc thủ thuật
\item Export sang Excel với thông tin chi tiết
\item Bao gồm thông tin phẫu thuật và thuốc sử dụng
\item Load dữ liệu từ database và format Excel 
\end{DoxyItemize}
