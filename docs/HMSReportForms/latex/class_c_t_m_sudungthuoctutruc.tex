\doxysubsection{Mô tả chi tiết}
Lớp view báo cáo sử dụng thuốc tự trực tại khoa. 

Class này kế thừa từ CGui\+View, cung cấp báo cáo về việc sử dụng thuốc tự trực\+:
\begin{DoxyItemize}
\item Lọc theo kho, đối tượng, loại thuốc
\item Lọc theo khoảng thời gian
\item Hỗ trợ lọc theo phẫu thuật và thủ thuật
\item Print preview và export dữ liệu
\item Load dữ liệu từ database và hiển thị báo cáo 
\end{DoxyItemize}
